two pointers and partition array 学习笔记

Mar 23 2020 Notes>algorithm 29 minutes read (About 4356 words)

这周学习双指针算法。 涉及该算法的题有以下几种类型。

  • 相向双指针。比如说求数组中第k大的数，此类题套用quick select 模板即可。
  • 同向双指针。一般也叫 sliding window，这类题比较难，一般套用 sliding window 模板。
Lintcode 148. sort colors

Based on value $O({N})$:

A: counting sort (计数排序) == hash map, radit sort, bucket sort …

Based on comparsion $O(N\log N)$:

B: quick sort , merge sort, bubble sort, insertion sort …

Java: comparator; python: cmp; …

  • 基于比较的排序，时间复杂度最优是 $O(N\log N)$
  • quick sort, merge sort 最重要
  • Lintcode 587. 去重应该在设计算法的时候就考虑，不要先求出所有的解再去重，这样的话可能花费大量的时间。比如说在1000个一里面找出三个相加等于三的三元组。

  24. Leetcode 969. Pancake Sorting

    容易理解 但是细节很多 不好掌握

    思路：(num[i], k) 这个pair代表原来数组中的一个数。i 表示nums[i] 在原数组中的位置（从0开始），k 是这个数当前的位置（排序后）。如果 k != i，那么做两次交换 [k + 1, i + 1] 即可把nums[i]变到它应该在的位置，与此同时更新这两次变换后对其他数的位置的影响。以此类推即可得到整个数组的排序 。

    Runtime: 48 ms, faster than 24.24% of Python3 online submissions for Pancake Sorting.

    Memory Usage: 13.9 MB, less than 10.00% of Python3 online submissions for Pancake Sorting.
